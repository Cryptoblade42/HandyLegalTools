Edited Books Citation Generator

For citing a chapter in an edited book (ie a work written by many authors that has been compiled into a single book)

Required Information:















Optional Information:



If there are multiple editors, type them all into the Editor field separated by a comma (except for the last one, which should use 'and') and add change 'ed' to 'eds'. See r 6.6.1 of the AGLC4 for examples.

Note that 'the' and abbreviations related to the publisher's corporate status (words like 'Pty', 'Co', etc) should be omitted. Locations ('Australia', 'Asia') should also be ommitted unless it's a crucial part of the publisher's name or is necessary to avoid ambiguity. Publication subdivisions should also be ignored. For examples, see r 6.3.1 of the AGLC4.
